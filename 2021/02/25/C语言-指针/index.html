жҢҮй’Ҳзҡ„еҹәжң¬жҰӮеҝө

  • ==жҢҮй’Ҳе°ұжҳҜең°еқҖпјҢиҖҢжҢҮй’ҲеҸҳйҮҸе°ұжҳҜеӯҳеӮЁең°еқҖзҡ„еҸҳйҮҸ==гҖӮ
  • жҢҮй’ҲеҸҳйҮҸзҡ„еЈ°жҳҺпјҡеңЁжҷ®йҖҡеҸҳйҮҸеЈ°жҳҺзҡ„еҹәзЎҖдёҠеҠ дёҠ*:
    • еҰӮжҢҮеҗ‘intзұ»еһӢеҜ№иұЎзҡ„жҢҮй’ҲеҸҳйҮҸпјҡint *p;
  • еҸ–ең°еқҖиҝҗз®—з¬Ұпјҡз”ЁдәҺиҺ·еҸ–еҸҳйҮҸзҡ„ең°еқҖпјҢжқҘз»ҷжҢҮй’ҲеҸҳйҮҸиөӢеҖјгҖӮ
    • зӨәдҫӢпјҡ
  • й—ҙжҺҘеҜ»еқҖиҝҗз®—з¬Ұпјҡз”ЁдәҺи®ҝй—®жҢҮй’ҲеҸҳйҮҸжҢҮеҗ‘зҡ„еҜ№иұЎзҡ„еҶ…е®№гҖӮ
    • зӨәдҫӢпјҡ
    • дёҚиҰҒж··ж·Ҷй—ҙжҺҘеҜ»еқҖиҝҗз®—з¬Ұе’ҢжҢҮй’ҲеҸҳйҮҸзҡ„еЈ°жҳҺж—¶зҡ„*зҡ„еҠҹиғҪгҖӮ
    • *pзӣёеҪ“дәҺiзҡ„==еҲ«еҗҚ==пјҢеҜ№*pжү§иЎҢзҡ„д»»дҪ•ж“ҚдҪңпјҢдёҺзӣҙжҺҘеҜ№iеҒҡж“ҚдҪңзҡ„ж•ҲжһңдёҖж ·гҖӮ

жҢҮй’ҲдёҺж•°з»„

  • жҢҮй’ҲеҸҜд»ҘжҢҮеҗ‘ж•°з»„е…ғзҙ :
    • зӨәдҫӢпјҡ
    • еҰӮжһң==жҢҮй’ҲжҢҮеҗ‘ж•°з»„е…ғзҙ пјҢйӮЈд№ҲеҜ№иҜҘжҢҮй’ҲиҝӣиЎҢз®—иҝҗз®—==е°ұжҳҜжңүж„Ҹд№үзҡ„пјҡ
      1. жҢҮй’ҲеҠ дёҠж•ҙж•°пјҡжҢҮй’ҲеҠ дёҠж•ҙж•°пјҢд»ЈиЎЁиҜҘжҢҮй’ҲжҢҮеҗ‘жӣҙеҸіиҫ№ж•°з»„зҡ„е…ғзҙ пјҢеҰӮ int *p = &a[0]пјҢз»ҷpеҠ дёҠ2д№ӢеҗҺпјҢpжҢҮеҗ‘a[2]
      2. жҢҮй’ҲеҮҸеҺ»ж•ҙж•°пјҡжҢҮй’ҲеҮҸеҺ»ж•ҙж•°пјҢд»ЈиЎЁиҜҘжҢҮй’ҲжҢҮеҗ‘жӣҙе·Ұиҫ№ж•°з»„зҡ„е…ғзҙ пјҢеҰӮint *p = a[2]пјҢз»ҷpеҮҸеҺ»2д№ӢеҗҺпјҢpжҢҮеҗ‘a[0]
      3. дёӨдёӘжҢҮй’ҲзӣёеҮҸпјҡдёӨдёӘжҢҮй’ҲзӣёеҮҸпјҢз»“жһңдёәиҝҷдёӨдёӘжҢҮй’ҲжҢҮеҗ‘зҡ„ж•°з»„е…ғзҙ д№Ӣй—ҙй—ҙйҡ”зҡ„ж•°з»„е…ғзҙ дёӘж•°пјҢеҰӮint *p = &a[5]; int *q = &a[1]пјҢйӮЈд№Ҳpе’ҢqзӣёеҮҸзҡ„з»“жһңе°ұжҳҜ4гҖӮ
  • еҸҜд»ҘзӣҙжҺҘе°Ҷж•°з»„еҗҚиөӢеҖјз»ҷжҢҮй’ҲеҸҳйҮҸпјҡ
    • еҰӮпјҡint a[2] = {0, 1, 2}; int *p = a;
    • е®һиҙЁжҳҜpжҢҮеҗ‘иҜҘж•°з»„зҡ„з¬¬дёҖдёӘе…ғзҙ a[0]
    • ==ж•°з»„зҡ„еҗҚеӯ—е®һиҙЁдёҠжҳҜжҢҮеҗ‘ж•°з»„з¬¬дёҖдёӘе…ғзҙ зҡ„жҢҮй’Ҳ==
    • дёҚиғҪз»ҷж•°з»„еҗҚиөӢеҖјгҖӮ
  • ж•°з»„еһӢеҪўејҸеҸӮж•°пјҡ
    • еңЁе°Ҷж•°з»„дҪңдёәеҪўејҸеҸӮж•°дј йҖ’з»ҷеҮҪж•°зҡ„ж—¶еҖҷпјҢе®һйҷ…дј йҖ’зҡ„жҳҜ==жҢҮеҗ‘иҜҘж•°з»„зҡ„з¬¬дёҖдёӘе…ғзҙ зҡ„жҢҮй’ҲпјҲжҲ–иҖ…иҜҙжҳҜж•°з»„еҗҚпјү==гҖӮ
    • ==еЈ°жҳҺеҪўејҸеҸӮж•°дёәжҢҮй’Ҳе°ұзӣёеҪ“дәҺеЈ°жҳҺеҸҳйҮҸдёәж•°з»„==(дҪҶжҳҜжҷ®йҖҡеҸҳйҮҸдёҚеҸҜд»Ҙиҝҷж ·зҗҶи§ЈпјүпјҢеҸҜд»Ҙ==е°Ҷж•°з»„еһӢеҪўејҸеҸӮж•°еЈ°жҳҺдёәжҢҮй’Ҳ==пјҢеҰӮпјҡint function(int a[], int n)дёҺint function(int *a, int n)зӯүд»·гҖӮ
    • еҸҜд»Ҙз»ҷеҮҪж•°дј йҖ’==ж•°з»„зҡ„зүҮж®ө==пјҢеҰӮпјҡfunction(&a[6], 10)
  • з”ЁжҢҮй’ҲдҪңдёәж•°з»„еҗҚпјҡ
    • еҸҜд»Ҙе°ҶжҢҮй’ҲзңӢеҒҡж•°з»„еҗҚиҝӣиЎҢ==еҸ–дёӢж Үж“ҚдҪң==гҖӮ
    • зј–иҜ‘еҷЁе°Ҷp[i]зңӢеҒҡ`*(p+i)гҖӮ
    • еҜ№жҢҮй’ҲиҝӣиЎҢеҸ–дёӢж Үж“ҚдҪңзҡ„еүҚжҸҗжҳҜ:==иҜҘжҢҮй’ҲжҳҜжҢҮеҗ‘дәҶдёҖдёӘж•°з»„зҡ„йҰ–е…ғзҙ ==гҖӮ
    • зӨәдҫӢпјҡ
  • жҢҮй’Ҳж•°з»„пјҡ
    • е…ғзҙ жҳҜжҢҮй’Ҳзҡ„ж•°з»„пјҢеҰӮпјҡint *p[10]пјҢеЈ°жҳҺдәҶдёҖдёӘж•°з»„пјҢиҝҷдёӘж•°з»„еӯҳеӮЁ10дёӘintеһӢжҢҮй’ҲеҸҳйҮҸгҖӮ
  • жҢҮеҗ‘жҢҮй’Ҳзҡ„жҢҮй’Ҳпјҡ
    • зӨәдҫӢпјҡ

жҢҮй’Ҳе’ҢеӨҡз»ҙж•°з»„

  • CиҜӯиЁҖдёӯзҡ„еӨҡз»ҙж•°з»„пјҢеҸҜд»ҘзҗҶи§Јдёәе°ҶжҜҸдёҖиЎҢзҡ„е…ғзҙ дҫқж¬ЎжҺ’еҲ—пјӣ
  • е°ҶдәҢз»ҙж•°з»„зңӢеҒҡжҳҜдёҖз»ҙж•°з»„жқҘеӨ„зҗҶпјҡ
    • жҢҮй’ҲжҢҮеҗ‘дәҢз»ҙж•°з»„зҡ„з¬¬дёҖдёӘе…ғзҙ пјҢжҢҮй’ҲжҜҸж¬ЎеҠ дёҖпјҢйғҪдјҡжҢҮеҗ‘ж•°з»„зҡ„дёӢдёҖдёӘе…ғзҙ пјҢеҰӮжһңеҲ°иҫҫжң¬иЎҢжң«е°ҫпјҢе°ұжҢҮеҗ‘дёӢдёҖиЎҢзҡ„з¬¬дёҖдёӘе…ғзҙ гҖӮ
  • е°ҶжҢҮй’ҲжҢҮеҗ‘жҹҗдёҖиЎҢзҡ„з¬¬дёҖдёӘе…ғзҙ пјҡ
    • дёҖиҲ¬ж–№жі•пјҡ
    • з®ҖеҶҷпјҡж №жҚ®еҸҜд»ҘеҜ№жҢҮй’ҲиҝӣиЎҢеҺ»дёӢж Үж“ҚдҪңпјҢеҲ©з”Ё&a[i][0] = &(*(a[i] + 0))
  • жҢҮеҗ‘дёҖз»ҙж•°з»„зҡ„жҢҮй’Ҳпјҡ
    • еҜ№дёҖз»ҙж•°з»„пјҢжҲ‘д»¬еёёз”Ёзҡ„жҢҮй’ҲжҳҜжҢҮеҗ‘==ж•°з»„зҡ„з¬¬дёҖдёӘе…ғзҙ зҡ„==пјӣ
    • еҸҜд»Ҙе®ҡд№ү==жҢҮеҗ‘дёҖдёӘдёҖз»ҙж•°з»„зҡ„жҢҮй’Ҳ==пјҡint (*p)[10],иҝҷжҳҜеЈ°жҳҺдәҶдёҖдёӘжҢҮеҗ‘й•ҝеәҰдёә10зҡ„ж•ҙеһӢж•°з»„зҡ„жҢҮй’Ҳpпјӣ
    • еҜ№дәҺдәҢз»ҙж•°з»„пјҢеҸҜд»ҘзңӢеҒҡжҳҜ==дёҖдёӘжҜҸдёӘе…ғзҙ йғҪжҳҜдёҖз»ҙж•°з»„зҡ„дёҖз»ҙж•°з»„==пјӣ
    • дҪҝз”ЁжҢҮеҗ‘ж•°з»„зҡ„жҢҮй’ҲжқҘ==жҢүиЎҢйҒҚеҺҶдәҢз»ҙж•°з»„==пјҡ
    • еңЁдәҢз»ҙж•°з»„дёӯпјҢa[i]еҸ–зҡ„жҳҜз¬¬i-1иЎҢз¬¬дёҖдёӘе…ғзҙ зҡ„ең°еқҖпјҢ&a[i]еҸ–зҡ„жҳҜз¬¬i-1иЎҢзҡ„ең°еқҖпјҲз¬¬i-1дёӘдёҖз»ҙж•°з»„зҡ„ең°еқҖпјү
    • еҜ№дәҺжҢҮеҗ‘ж•°з»„зҡ„жҢҮй’ҲжқҘиҜҙпјҡ*pжҳҜеҸ–еҮәжҢҮеҗ‘зҡ„ж•°з»„пјҢ(*p)[i]жҳҜеҸ–еҮәжҢҮеҗ‘зҡ„ж•°з»„дёӯзҡ„з¬¬i-1дёӘе…ғзҙ гҖӮ
  • з”ЁеӨҡз»ҙж•°з»„еҗҚдҪңдёәжҢҮй’Ҳпјҡ
    • дәҢз»ҙж•°з»„зҡ„==ж•°з»„еҗҚжҳҜдёҖдёӘжҢҮеҗ‘дёҖз»ҙж•°з»„зҡ„жҢҮй’Ҳ==пјҢдҫӢеҰӮint a[10][10]зҡ„ж•°з»„еҗҚaд»ЈиЎЁзҡ„ж„ҸжҖқжҳҜ&a[0]гҖӮ
  • еӨҡз»ҙж•°з»„зҡ„й•ҝеәҰпјҡ
    • еҜ№дәҺ==дёҖз»ҙж•°з»„еңЁеЈ°жҳҺж—¶е°ұиөӢеҖјзҡ„жғ…еҶөдёӢпјҢеҸҜд»ҘзңҒз•Ҙж•°з»„зҡ„й•ҝеәҰ==пјҢеҰӮпјҡint a[] = {1, 2}
    • ==дәҢз»ҙж•°з»„еңЁеЈ°жҳҺж—¶е°ұиөӢеҖјеҸҜд»ҘзңҒз•Ҙз¬¬дёҖз»ҙзҡ„й•ҝеәҰ==,жңҖеҗҺзҡ„дәҢз»ҙж•°з»„зҡ„жҜҸдёҖз»ҙзҡ„й•ҝеәҰйғҪжҳҜжҜҸдёҖиЎҢзҡ„ж•°з»„зҡ„й•ҝеәҰжҳҜ==ж №жҚ®жңҖй•ҝзҡ„ж•°з»„зҡ„й•ҝеәҰжқҘзЎ®е®ҡ==зҡ„гҖӮеҰӮжһңжңүдёҖдәӣиЎҢзҡ„ж•°жҚ®дёҚеӨҹеЎ«ж»Ўж•ҙиЎҢпјҢйӮЈд№ҲCиҜӯиЁҖ==дјҡз”Ёз©әеӯ—з¬Ұ\0жқҘеЎ«иЎҘ==гҖӮ

еӯ—з¬ҰдёІж•°з»„

дәҢз»ҙж•°з»„е®һзҺ°еӯ—з¬ҰдёІж•°з»„

  • еӯ—з¬ҰдёІеӯ—йқўйҮҸпјҡ
    • еӯ—з¬ҰдёІжҳҜдёҖдёӘ==еӯ—з¬Ұж•°з»„==пјӣ
    • еӯ—з¬ҰдёІеӯ—йқўйҮҸе°ұжҳҜиҝҷдёӘж•°з»„зҡ„ж•°з»„еҗҚпјҢ==жҳҜдёҖдёӘchar *зұ»еһӢзҡ„жҢҮй’Ҳ==гҖӮ
    • еӯ—з¬ҰдёІеӯ—йқўйҮҸжҳҜ==дёҚиғҪиў«дҝ®ж”№зҡ„==
  • еҲӣе»әдёҖдёӘдәҢз»ҙж•°з»„пјҢжҢүз…§жҜҸиЎҢдёҖдёӘеӯ—з¬ҰдёІ(==дёҖз»ҙеӯ—з¬Ұж•°з»„==)зҡ„ж–№ејҸпјҢе°ҶдёҖзі»еҲ—еӯ—з¬ҰдёІеӯҳе…ҘдёҖдёӘж•°з»„дёӯгҖӮ
  • зӨәдҫӢпјҡ
  • еӣ дёәжҜҸдёҖиЎҢеӯ—з¬ҰдёІзҡ„й•ҝеәҰжҳҜзңҒз•Ҙзҡ„пјҢжүҖд»ҘжҜҸдёҖиЎҢзҡ„й•ҝеәҰжҳҜзі»з»ҹиҮӘеҠЁзЎ®е®ҡзҡ„гҖӮеңЁдәҢз»ҙж•°з»„дёӯпјҢжҜҸдёҖиЎҢзҡ„ж•°з»„зҡ„й•ҝеәҰжҳҜж №жҚ®жңҖй•ҝзҡ„ж•°з»„зҡ„й•ҝеәҰжқҘзЎ®е®ҡзҡ„гҖӮеҰӮжһңжңүдёҖдәӣиЎҢзҡ„ж•°жҚ®дёҚеӨҹеЎ«ж»Ўж•ҙиЎҢпјҢйӮЈд№ҲCиҜӯиЁҖдјҡз”Ёз©әеӯ—з¬Ұ\0жқҘеЎ«иЎҘпјҢйҖ жҲҗдәҶз©әй—ҙжөӘиҙ№гҖӮ
  • еҸҜд»ҘзҗҶи§Јдёәз”Ёиҝҷз§Қж–№ејҸеҲӣе»әзҡ„дәҢз»ҙж•°з»„йғҪжҳҜзҹ©еҪўзҡ„гҖӮ

жҢҮй’Ҳж•°з»„е®һзҺ°еӯ—з¬ҰдёІж•°з»„

  • иҰҒжғіе®һзҺ°еҸӮе·®дёҚйҪҗзҡ„дәҢз»ҙж•°з»„пјҢйңҖиҰҒдҪҝз”Ёе…ғзҙ дёәжҢҮй’Ҳзҡ„ж•°з»„гҖӮ
  • е»әз«ӢдёҖдёӘе…ғзҙ йғҪжҳҜжҢҮеҗ‘еӯ—з¬ҰдёІзҡ„жҢҮй’Ҳзҡ„ж•°з»„пјҢжқҘе®һзҺ°еӯ—з¬ҰдёІж•°з»„гҖӮ
  • зӨәдҫӢпјҡ

еӯ—з¬ҰдёІж•°з»„зҡ„еә”з”Ёпјҡе‘Ҫд»ӨиЎҢеҸӮж•°

  • иҝҗиЎҢзЁӢеәҸжҳҜйңҖиҰҒжҸҗдҫӣдёҖдәӣдҝЎжҒҜпјҢиҝҷдәӣдҝЎжҒҜд»Һе‘Ҫд»ӨиЎҢдёӯжҸҗдҫӣпјҢз§°дёәе‘Ҫд»ӨиЎҢеҸӮж•°пјҲCиҜӯиЁҖдёӯд№ҹеҸ«зЁӢеәҸеҸӮж•°пјүгҖӮ
  • дёәдәҶи®ҝй—®е‘Ҫд»ӨиЎҢеҸӮж•°еҝ…йЎ»е°ҶmainеҮҪж•°е®ҡд№үдёәеҗ«жңүдёӨдёӘзү№ж®ҠеҸӮж•°зҡ„зҡ„еҮҪж•°пјҡ
    • argcеҸӮж•°пјҡargcжҳҜеҸӮж•°и®Ўж•°зҡ„еҸӮж•°пјҢдёәintеһӢпјҢз”ЁдәҺи®°еҪ•е‘Ҫд»ӨиЎҢеҸӮж•°зҡ„ж•°йҮҸгҖӮ
    • argvеҸӮж•°пјҡargvжҳҜжҢҮеҗ‘е‘Ҫд»ӨиЎҢеҸӮж•°зҡ„жҢҮй’Ҳж•°з»„пјҢиҝҷдәӣе‘Ҫд»ӨиЎҢеҸӮж•°д»Ҙеӯ—з¬ҰдёІзҡ„еҪўејҸеӯҳеӮЁгҖӮ
      • argvжҳҜchar *argv[]еһӢзҡ„пјҢе®һиҙЁе°ұжҳҜдёҖдёӘеӯ—з¬ҰдёІж•°з»„гҖӮ
      • argv[0]жҢҮеҗ‘зЁӢеәҸеҗҚпјӣ
      • argv[1]еҲ°argv[argc - 1]жҢҮеҗ‘дҪҷдёӢзҡ„е‘Ҫд»ӨиЎҢеҸӮж•°гҖӮ
      • argv[argc]жҳҜдёҖдёӘйҷ„еҠ е…ғзҙ пјҢиҝҷдёӘе…ғзҙ е§Ӣз»ҲжҳҜдёҖдёӘз©әжҢҮй’ҲгҖӮ
  • зӨәдҫӢпјҡ

еҠЁжҖҒеӯҳеӮЁеҲҶй…Қ

  • еҠЁжҖҒеӯҳеӮЁеҲҶй…ҚпјҡеңЁзЁӢеәҸ==иҝҗиЎҢжңҹй—ҙ==еҲҶй…ҚеҶ…еӯҳеҚ•е…ғзҡ„иғҪеҠӣ
  • еҶ…еӯҳеҲҶй…ҚеҮҪж•°жүҖиҺ·еҫ—зҡ„еҶ…еӯҳеқ—йғҪжқҘиҮӘдёҖдёӘз§°дёә**е ҶпјҲheapпјү**зҡ„еӯҳеӮЁжұ гҖӮ
  • еҶ…еӯҳеҲҶй…ҚеҮҪж•°жңүдёүз§ҚпјҢйғҪжҳҜеЈ°жҳҺеңЁ<stdlib.h>еӨҙдёӯзҡ„пјҡ
    1. mallocеҮҪж•°пјҡеҲҶй…ҚеҶ…еӯҳеқ—пјҢдҪҶжҳҜдёҚеҜ№еҶ…еӯҳеқ—иҝӣиЎҢеҲқе§ӢеҢ–гҖӮпјҲжңҖеёёз”ЁпјҢжңҖй«ҳж•Ҳпјү
      • еҺҹеһӢпјҡvoid *malloc(size_t size),mallocеҮҪж•°еҲҶй…ҚsizeдёӘеӯ—иҠӮзҡ„еҶ…еӯҳеқ—пјҢе№¶дё”иҝ”еӣһжҢҮеҗ‘иҜҘеҶ…еӯҳеқ—зҡ„жҢҮй’ҲгҖӮ
      • зӨәдҫӢпјҡint *a = (n * sizeof(int));,дёәй•ҝеәҰдёәnзҡ„intеһӢж•°з»„еҲҶй…Қз©әй—ҙгҖӮ
    2. callocеҮҪж•°пјҡеҲҶй…ҚеҶ…еӯҳеқ—пјҢе№¶дё”еҜ№еҶ…еӯҳеқ—иҝӣиЎҢжё…йӣ¶гҖӮ
      • еҺҹеһӢпјҡvoid *calloc(size_t nmemb, size_t size);,callocеҮҪж•°дёәnmembдёӘе…ғзҙ зҡ„ж•°з»„еҲҶй…ҚеҶ…еӯҳз©әй—ҙпјҢе…¶дёӯжҜҸдёӘе…ғзҙ зҡ„й•ҝеәҰйғҪжҳҜsizeдёӘеӯ—иҠӮгҖӮ
      • еңЁеҲҶй…ҚдәҶеҶ…еӯҳд№ӢеҗҺпјҢcallocеҮҪж•°дјҡ==йҖҡиҝҮжҠҠжүҖжңүдҪҚи®ҫзҪ®дёә0зҡ„ж–№ејҸиҝӣиЎҢеҲқе§ӢеҢ–==гҖӮ
      • е°Ҷnemebи®ҫдёә1пјҢеҸҜд»Ҙдёә==д»»дҪ•зұ»еһӢзҡ„ж•°жҚ®йЎ№еҲҶй…ҚеҶ…еӯҳ==гҖӮ
      • зӨәдҫӢпјҡint *a = calloc(n, sizeof(int))пјҢдёәй•ҝеәҰдёәnзҡ„intеһӢж•°з»„еҲҶй…Қз©әй—ҙгҖӮ
    3. reallocеҮҪж•°пјҡи°ғж•ҙе…ҲеүҚеҲҶй…Қзҡ„еҶ…еӯҳеқ—еӨ§е°ҸгҖӮ
      • еҺҹеһӢпјҡvoid *realloc(void *ptr, size_t size);,и°ғз”ЁreallocеҮҪж•°ж—¶пјҢ==ptrеҝ…йЎ»жҢҮеҗ‘е…ҲеүҚйҖҡиҝҮmallocпјҢcallocжҲ–reallocзҡ„и°ғз”ЁиҺ·еҫ—зҡ„еҶ…еӯҳеқ—==гҖӮsizeиЎЁзӨәеҶ…еӯҳеқ—зҡ„ж–°е°әеҜёгҖӮ
      • еҰӮжһңrealloc==д»Ҙз©әжҢҮй’ҲдҪңдёәз¬¬дёҖдёӘе®һйҷ…еҸӮж•°пјҢйӮЈд№Ҳе®ғзҡ„иЎҢдёәе°ұиЎҢmallocеҮҪж•°дёҖж ·==гҖӮ
      • еҰӮжһңrealloc==д»Ҙ0дҪңдёәз¬¬дәҢдёӘе®һйҷ…еҸӮж•°пјҢйӮЈд№Ҳе®ғдјҡйҮҠж”ҫжҺүеҶ…еӯҳеқ—==гҖӮ
  • еҶ…еӯҳеҲҶй…ҚеҮҪж•°зҡ„иҝ”еӣһеҖјпјҡ
    • void *:еҶ…еӯҳеҲҶй…ҚеҮҪж•°дјҡиҝ”еӣһvoid *зұ»еһӢзҡ„еҖјпјҢиҜҘзұ»еһӢзҡ„жҢҮй’ҲжҳҜ==йҖҡз”ЁжҢҮй’Ҳ==пјҢжҢҮеҗ‘еҲҶй…Қзҡ„еҶ…еӯҳгҖӮ
      • еҸҜд»Ҙе°Ҷ==void *зұ»еһӢзҡ„еҸҳйҮҸиөӢз»ҷд»»ж„Ҹзұ»еһӢзҡ„жҢҮй’ҲеҸҳйҮҸгҖӮ
  • з©әжҢҮй’Ҳпјҡ
    • еҪ“жүҫдёҚеҲ°йңҖиҰҒзҡ„и¶іеӨҹеӨ§зҡ„еҶ…еӯҳеқ—ж—¶пјҢеҶ…еӯҳеҲҶй…ҚеҮҪж•°дјҡиҝ”еӣһ==з©әжҢҮй’Ҳ==пјӣ
    • з©әжҢҮй’ҲжҳҜ==дёҚжҢҮеҗ‘д»»дҪ•ең°ж–№зҡ„жҢҮй’Ҳ==пјҢиҝҷжҳҜдёҖдёӘз”ЁдәҺеҢәеҲ«дәҺжүҖжңүжңүж•ҲжҢҮй’Ҳзҡ„зү№ж®ҠеҖјгҖӮ
    • жөӢиҜ•еҶ…еӯҳеҲҶй…ҚеҮҪж•°зҡ„иҝ”еӣһеҖјжҳҜеҗҰжҳҜз©әжҢҮй’ҲпјҢз”ЁжқҘ==еҲӨж–ӯеҶ…еӯҳеҲҶй…ҚжҳҜеҗҰжҲҗеҠҹ==гҖӮ
    • NULLе®Ҹпјҡз©әжҢҮй’Ҳз”ЁNULLе®ҸжқҘиЎЁзӨәгҖӮNULLе®ҸеңЁеӨҡдёӘеӨҙпјҲеҢ…жӢ¬<stido.h>е’Ң<stdlib.h>пјүдёӯйғҪжңүе®ҡд№үгҖӮ
  • йҮҠж”ҫеӯҳеӮЁз©әй—ҙпјҡ
    • free()еҮҪж•°пјҡ
      • еҺҹеһӢпјҡvoid free(void *ptr)
      • ==и°ғз”ЁfreeеҮҪж•°дјҡйҮҠж”ҫжҺүptrжҢҮй’ҲжҢҮеҗ‘зҡ„еҶ…еӯҳеқ—==
      • freeеҮҪж•°зҡ„е®һйҷ…еҸӮж•°еҝ…йЎ»жҳҜ==е…ҲеүҚз”ұеҶ…еӯҳеҲҶй…ҚеҮҪж•°иҝ”еӣһзҡ„жҢҮй’Ҳ==гҖӮ

жҢҮеҗ‘еҮҪж•°зҡ„жҢҮй’Ҳ

  • еҰӮжһңеңЁзЁӢеәҸдёӯе®ҡд№үдәҶдёҖдёӘеҮҪж•°пјҢйӮЈд№ҲеңЁзј–иҜ‘ж—¶зі»з»ҹе°ұдјҡдёәиҝҷдёӘеҮҪж•°д»Јз ҒеҲҶй…ҚдёҖж®өеӯҳеӮЁз©әй—ҙпјҢиҝҷж®ө==еӯҳеӮЁз©әй—ҙзҡ„йҰ–ең°еқҖ==з§°дёәиҝҷдёӘеҮҪж•°зҡ„ең°еқҖ,иҖҢдё”==еҮҪж•°еҗҚиЎЁзӨәзҡ„е°ұжҳҜиҝҷдёӘең°еқҖ==гҖӮ
  • еҮҪж•°жҢҮй’Ҳпјӣз”ЁдёҖдёӘжҢҮй’ҲеҸҳйҮҸжқҘеӯҳеӮЁеҮҪж•°зҡ„ең°еқҖгҖӮ
  • еҮҪж•°жҢҮй’Ҳзҡ„еЈ°жҳҺпјҡ
    • еЈ°жҳҺжЁЎжқҝпјҡеҮҪж•°иҝ”еӣһеҖјзұ»еһӢ (*еҮҪж•°жҢҮй’ҲеҸҳйҮҸеҗҚ)(еҮҪж•°еҸӮж•°зұ»еһӢеҲ—иЎЁ);
    • зӨәдҫӢпјҡint (*p)(int,int);
  • еҮҪж•°жҢҮй’Ҳи°ғз”ЁеҮҪж•°пјҡ
    • йҖҡиҝҮ*иҝҗз®—з¬ҰжқҘеҸ–еҮәе®һйҷ…зҡ„еҮҪж•°гҖӮд№ҹеҸҜд»ҘзӣҙжҺҘйҖҡиҝҮеҮҪж•°жҢҮй’ҲеҸҳйҮҸеҗҚи°ғз”Ёпјӣ
    • зӨәдҫӢпјҡ
  • еӯҳеӮЁеҮҪж•°жҢҮй’Ҳзҡ„жҢҮй’Ҳж•°з»„пјҡ
    • еҸҜд»ҘеЈ°жҳҺе…ғзҙ дёәеҮҪж•°жҢҮй’Ҳзҡ„ж•°з»„пјҡvoid (*funs[10])(int);
